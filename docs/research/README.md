# Research Notes

This directory is for public-safe research summaries and model evaluation reports.

Use it for:

- Model suitability evaluations.
- Benchmark methodology notes.
- Product tradeoff reports.
- Reproducible environment notes.

Do not commit:

- Raw user media.
- Generated transcripts from private media.
- Full logs containing local paths or environment details.
- Model weights or runtime directories.
- Large benchmark outputs that belong in ignored work directories.

When adding a report, include:

- Date of evaluation.
- Model or service version.
- Source links.
- Hardware and platform.
- Whether the result is reproducible on Apple Silicon macOS.
- Quality, speed, memory or cost, timestamp behavior, privacy, platform, integration complexity, and license or commercial-use risks.
